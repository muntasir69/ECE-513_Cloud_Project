All About Our Project

The aim of the ECE 513 project is to create a device capable of tracking both heart rate and oxygen saturation. Additionally, a website will be developed to facilitate logins for both patients and physicians, enabling them to monitor the recorded readings. The specific project requirements are outlined in the reference section. Provided below is a brief video showcasing the heart monitor device, which utilizes an Argon microprocessor.
